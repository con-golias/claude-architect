## Learning Paths

### C# Path

```
1. Beginner: Head First C# → C# fundamentals, OOP
2. Intermediate: C# 12 and .NET 8 (Price) → modern C#, ASP.NET Core
3. Web: ASP.NET Core Minimal APIs → REST APIs, middleware
4. Data: Entity Framework Core → database access, migrations
5. Advanced: C# in Depth (Skeet) → language internals
6. Performance: Concurrency in C# → async patterns, parallel
7. Architecture: Clean Architecture with .NET → DDD, CQRS
```

### .NET Backend Path

```
1. Minimal APIs → basic HTTP endpoints
2. ASP.NET Core MVC → controllers, views, filters
3. Entity Framework Core → ORM, migrations, LINQ
4. Authentication/Authorization → Identity, JWT, OAuth
5. Background Services → Hosted services, Quartz, Hangfire
6. Messaging → MassTransit, Azure Service Bus
7. .NET Aspire → Cloud-native orchestration
8. Native AOT → Performance optimization
```
