## Learning Paths

### Dart Language Path

```
1. Beginner: Dart Language Tour → syntax, types, functions
2. Null safety: Sound Null Safety Guide → nullable types, flow analysis
3. Intermediate: Effective Dart → style, documentation, usage, design
4. Dart 3.0: Records, patterns, sealed classes → modern Dart
5. Async: Futures, Streams, Isolates → concurrent programming
6. Advanced: Extension types, macros → advanced features
```

### Flutter Development Path

```
1. Setup: Flutter installation, first app → Flutter basics
2. Widgets: StatelessWidget, StatefulWidget, layout → widget tree
3. Navigation: go_router → routing, deep links
4. State: Riverpod or Bloc → state management
5. Data: dio + freezed → networking, models
6. Database: drift or Hive → local storage
7. Testing: Unit + Widget + Integration → quality
8. Architecture: Clean Architecture → scalable apps
9. Advanced: Custom painters, animations, platform channels
10. Release: App Store + Play Store deployment → publishing
```
