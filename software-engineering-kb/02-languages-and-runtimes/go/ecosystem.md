## Ecosystem Overview

Go's ecosystem is focused on **cloud infrastructure, networking, DevOps, and backend services**. The most critical infrastructure of the modern cloud runs on Go: Docker, Kubernetes, Terraform, Prometheus, and etcd.

### net/http Enhanced Routing (Go 1.22+)

```go
// Before 1.22: needed third-party router
mux := http.NewServeMux()

// Go 1.22+: method-aware, path parameters built-in
mux.HandleFunc("GET /users/{id}", getUser)
mux.HandleFunc("POST /users", createUser)
mux.HandleFunc("DELETE /users/{id}", deleteUser)
mux.HandleFunc("GET /files/{path...}", serveFile) // Wildcard

func getUser(w http.ResponseWriter, r *http.Request) {
    id := r.PathValue("id") // Built-in path parameter extraction
    // ...
}
```

## gRPC — Go as First-Class gRPC Language

Go is the **primary language** for gRPC development. Google's gRPC was designed with Go as a primary target.

```protobuf
// user.proto
service UserService {
  rpc GetUser(GetUserRequest) returns (User);
  rpc ListUsers(ListUsersRequest) returns (stream User); // server streaming
  rpc CreateUsers(stream User) returns (CreateUsersResponse); // client streaming
}
```

```go
// Generated server implementation
type server struct {
    pb.UnimplementedUserServiceServer
}

func (s *server) GetUser(ctx context.Context, req *pb.GetUserRequest) (*pb.User, error) {
    user, err := s.db.FindUser(ctx, req.Id)
    if err != nil {
        return nil, status.Errorf(codes.NotFound, "user not found")
    }
    return user, nil
}
```

### sqlc Pattern (Write SQL, Get Type-Safe Go)

```sql
-- query.sql
-- name: GetUser :one
SELECT id, name, email FROM users WHERE id = $1;

-- name: ListUsers :many
SELECT id, name, email FROM users ORDER BY name;

-- name: CreateUser :one
INSERT INTO users (name, email) VALUES ($1, $2) RETURNING *;
```

```go
// Generated by sqlc — fully type-safe
func (q *Queries) GetUser(ctx context.Context, id int64) (User, error)
func (q *Queries) ListUsers(ctx context.Context) ([]User, error)
func (q *Queries) CreateUser(ctx context.Context, arg CreateUserParams) (User, error)
```
