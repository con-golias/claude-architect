## Learning Paths

### Java Path

```
1. Beginner: Head First Java → Java basics, OOP
2. Intermediate: Effective Java → Best practices
3. Spring: Spring in Action → Spring Boot
4. Advanced: Java Concurrency in Practice → Virtual threads
5. Architecture: Microservices patterns with Spring Cloud
```

### Kotlin Path

```
1. Beginner: Kotlin Koans (kotlinlang.org/docs/koans.html)
2. Intermediate: Kotlin in Action
3. Coroutines: Official coroutines guide → Kotlin Coroutines (book)
4. Android: Jetpack Compose → Android development
5. KMP: Kotlin Multiplatform → cross-platform
```
