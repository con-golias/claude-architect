## Essential Books

### JavaScript

| Book | Author | Level | Focus | Why Read It |
|------|--------|-------|-------|-------------|
| **"You Don't Know JS Yet" (YDKJS)** | Kyle Simpson | Intermediate-Advanced | JS internals | Deepest understanding of JS mechanics: scope, closures, this, prototypes, async |
| **"JavaScript: The Good Parts"** | Douglas Crockford | Beginner-Intermediate | Language subset | Classic that shaped modern JS practices (somewhat dated but foundational) |
| **"Eloquent JavaScript"** | Marijn Haverbeke | Beginner | Full introduction | Free online, excellent for beginners, covers DOM and Node.js |
| **"JavaScript: The Definitive Guide"** | David Flanagan | All levels | Comprehensive reference | 7th edition covers ES2020+, thorough and authoritative |
| **"High Performance JavaScript"** | Nicholas Zakas | Advanced | Performance | V8 optimization, DOM performance, network optimization |
| **"JavaScript Patterns"** | Stoyan Stefanov | Intermediate | Design patterns | Gang of Four patterns adapted for JavaScript |

### TypeScript

| Book | Author | Level | Focus | Why Read It |
|------|--------|-------|-------|-------------|
| **"Effective TypeScript"** | Dan Vanderkam | Intermediate-Advanced | 83 specific items | Best practices in the style of "Effective Java" — most impactful TS book |
| **"Programming TypeScript"** | Boris Cherny | Intermediate | Type system | Deep dive into type system, advanced types, type-level programming |
| **"TypeScript in 50 Lessons"** | Stefan Baumgartner | Beginner-Intermediate | Progressive learning | Well-structured progression from basics to advanced |
| **"Learning TypeScript"** | Josh Goldberg | Beginner | Introduction | Modern introduction with practical examples |
| **"Type-Level TypeScript"** | Gabriel Vergnaud | Advanced | Type-level programming | Template literal types, recursive types, type challenges |

## Learning Paths

### Beginner → Intermediate (3-6 months)

```
1. JavaScript Basics
   └── javascript.info (Parts 1-2)
   └── Eloquent JavaScript (Chapters 1-10)

2. DOM & Browser APIs
   └── MDN Web Docs tutorials
   └── javascript.info (Part 3)

3. TypeScript Fundamentals
   └── TypeScript Handbook (official)
   └── Execute Program (interactive)

4. First Framework
   └── React: react.dev (new docs)
   └── Vue: vuejs.org/tutorial
   └── Svelte: learn.svelte.dev
```

### Intermediate → Advanced (6-12 months)

```
1. Deep JavaScript
   └── "You Don't Know JS Yet" book series
   └── Event loop, closures, prototypes mastery

2. Advanced TypeScript
   └── "Effective TypeScript" by Dan Vanderkam
   └── Total TypeScript workshops
   └── Type Challenges (github.com/type-challenges)

3. Node.js Mastery
   └── "Node.js Design Patterns" book
   └── Streams, worker threads, clustering

4. Architecture & Patterns
   └── Full-stack with Next.js/Remix/SvelteKit
   └── Testing (Vitest + Testing Library + Playwright)
   └── Performance optimization
```

### Expert Level (Ongoing)

```
1. Engine Internals
   └── V8 blog posts
   └── Browser rendering pipeline

2. Compiler/Tooling
   └── Build your own bundler/parser
   └── AST manipulation (Babel/SWC plugins)

3. Community Contribution
   └── TC39 proposal tracking
   └── Open source contribution
   └── Conference speaking
```
