## Learning Paths

### Beginner → Intermediate (2-4 months)

```
1. The Rust Programming Language (The Book)
   └── Read chapters 1-12 carefully

2. Rustlings exercises
   └── Complete all exercises (reinforces The Book)

3. Practice projects
   └── CLI tool with clap, grep clone, simple web server

4. Rust by Example
   └── Fill in any gaps from The Book
```

### Intermediate → Advanced (3-6 months)

```
1. "Rust for Rustaceans" by Jon Gjengset
   └── Advanced ownership, traits, unsafe

2. "Zero To Production In Rust"
   └── Production web services, testing, deployment

3. "Rust Atomics and Locks" by Mara Bos
   └── Low-level concurrency

4. Async Rust mastery
   └── Tokio tutorial, async patterns, Pin/Future
```

### Expert (Ongoing)

```
1. The Rustonomicon (unsafe Rust)
2. Contribute to open source Rust projects
3. Read compiler source code
4. Study LLVM optimization passes
5. Embedded Rust, no_std programming
```
