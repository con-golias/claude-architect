## Learning Paths

### Swift Language Path

```
1. Beginner: Swift Playgrounds app → interactive learning
2. Fundamentals: Hacking with Swift (100 Days of Swift) → comprehensive
3. Intermediate: Swift in Depth → patterns, protocols, generics
4. Concurrency: WWDC sessions → async/await, actors, structured concurrency
5. Advanced: Advanced Swift (objc.io) → language internals
6. Expert: Swift Evolution proposals → language design decisions
```

### iOS Development Path

```
1. Swift basics: Variables, control flow, functions, OOP
2. SwiftUI fundamentals: Views, modifiers, state management
3. Navigation: NavigationStack, TabView, sheets
4. Data: SwiftData, networking (URLSession), JSON (Codable)
5. Architecture: MVVM with @Observable, or TCA
6. Async: async/await, TaskGroup, actors
7. Testing: Swift Testing, XCUITest
8. Advanced: Custom components, animations, gestures
9. Distribution: App Store submission, TestFlight
10. Specialization: CoreML, ARKit, WidgetKit, visionOS
```
