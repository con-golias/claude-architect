# Privacy Impact Assessment

## 1. Data Inventory

### Personal Data Collected
- [ ] Name, email, phone
- [ ] Address, location data
- [ ] Financial information
- [ ] Health / biometric data
- [ ] Government IDs (SSN, passport)

### Data Sources
- **Direct collection**: Forms, registration, user input
- **Indirect collection**: Analytics, cookies, third-party APIs

## 2. Purpose & Legal Basis

| Data Category | Purpose | Legal Basis | Retention Period |
|---------------|---------|-------------|-----------------|
| Email         | Account auth | Consent | Until account deletion |
| Usage data    | Analytics | Legitimate interest | 12 months |

## 3. Data Protection Measures

- [ ] Encryption at rest (AES-256 or equivalent)
- [ ] Encryption in transit (TLS 1.2+)
- [ ] PII masking in logs and error reports
- [ ] Access control (role-based, least privilege)
- [ ] Data minimization (collect only what's needed)

## 4. Data Subject Rights

- [ ] Right to access (export user data)
- [ ] Right to rectification (edit personal data)
- [ ] Right to erasure (delete account and data)
- [ ] Right to portability (machine-readable export)
- [ ] Right to object (opt-out of processing)
